What You Need to Know

And perhaps some of what you don't

You will forget all the things

So remember some logical commands
                    
vi ~/.gitrc
                    
                    
alias   g='hub'
#Basic Git Commands
alias gb='g branch -vv'
alias gc='g checkout'
alias gpush='g push --follow-tags'
#Diff
alias gd='g diff'
alias gds='gd --staged'
                    
                    
vi ~/.zshrc
source ~/.gitrc

Git gets messy

So make cleanup easy

function listLocalBranches() {
  git fetch --prune && git branch -r |
    awk '{print $1}' |
    egrep -v -f /dev/fd/0 <(git branch -vv | grep origin) |
    awk '{print $1}' }

alias gdeletelocalbranches="listLocalBranches |
    xargs git branch -D"
Show how this happens by creating branches and deleting them elsewhere (GitHub)
